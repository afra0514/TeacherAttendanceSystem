Phase 1: Logic & Database Changes:

1.  Rename Database Tables:
      - Go to phpMyAdmin and rename the instructors table to match your new
        entity (e.g., students, employees, or products).
2.  Modify Table Columns:
      - Change column names. For example, replace designation with department,
        category, or batch.
3.  Refactor Folder & File Names:
      - Rename the folder /instructors/ to something like /employees/.
      - Rename internal files (e.g., view_instructor.php to view_employee.php).
4.  Update PHP Queries:
      - Update all SQL queries in your PHP files. Change SELECT * FROM
        instructors to SELECT * FROM employees.
5.  Update Status Logic:
      - Change the values of the status column in the attendance table. Instead
        of Present/Absent, use project-specific terms like Paid/Unpaid,
        Active/Inactive, or Completed/Pending.
6.  Modify UI Text:
      - Change all <h1> titles, <p> descriptions, and form placeholders to match
        the new project theme.

Phase 2: CSS Changes for a Unique Look:

Update these properties in your style.css to make each project look different:

  - Color Palette (Theming):
      - Change the --primary variable in :root.
      - Use Green (#27ae60) for Financial/Health systems.
      - Use Orange (#f39c12) for Creative/Food services.
      - Use Dark Blue/Black (#2c3e50) for Corporate/Law systems.
  - Typography (Fonts):
      - Change the Google Font import. Use 'Poppins' for a friendly UI, 'Roboto'
        for a tech look, or 'Playfair Display' for a luxury/elegant feel.
  - Border Radius (Shapes):
      - Currently, you are using 20px (Soft UI). Change it to 0px for a sharp
        professional look or 8px for a standard modern look.
  - Sidebar Theme:
      - Switch between a Light Sidebar (White background, dark text) and a Dark
        Sidebar (Dark background, white text). This completely changes the
        dashboard feel.
  - Shadows and Depth:
      - Adjust the --shadow variable. Use a heavy shadow for a floating
        "Material Design" look or thin borders (1px solid #ddd) with no shadow
        for a "Minimalist" look.
  - Navigation Profile Design:
      - Change the avatar shape. Switch from a Squircle (12px radius) to a
        Perfect Circle (50% radius).
  - Button Hover Effects:
      - Instead of moving the button up (translateY), try a glow effect
        (box-shadow change) or a background-slide effect.
  - Background Style:
      - Instead of a solid light color, use a very subtle gradient or a
        geometric background pattern on the body tag.

By following these steps, you can reuse the same high-quality code structure
for 30+ different projects while making each one look like it was built from
scratch!
